To run the program, please use one of the following methods:

1. Change the directory to the Product folder. Insert the following command into the Terminal/Command Prompt
javac -cp .:derby.jar Driver

2. Change the directory to the Product folder. Insert the following command into the Terminal/Command Prompt
java -jar GolfManager.jar

3. Double click on the jar file to open the project

The database already contains data. If you wish to delete the database, dispose of the Golf file into the trash and then enter the following command into the Terminal/Command Prompt. This will reinstall the database upon launch of the project:

javac -cp .:derby.jar Driver

If you wish to independently install the database without launching the program, please enter the following command into the Terminal/Command Prompt:

javac -cp .:derby.jar Install
